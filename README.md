## About

The goal is to give the user a deep dive into who David is and what his abilities are. All of the Top Projects are there with clean Documentation listed and you can also check out a CV by downloading it.



### local installation:

1. clone the repo

```
git clone https://github.com/DaniDevOfficial/Portfolio-Website
```

2. cd into cloned repo

```
cd Wishlingo
```

3. install dependencies

```
npm install
```

```
npm install react-router-dom
```

```
npm install @emailjs
```

```
npm install react-icons
```

```
npm install typewriter-effect
```

```
npm install react-toastify
```


4. run the app

```
npm run start
```
